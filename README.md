# Moodle マイコース改善デモ var.2

HCI設計論のグループ課題用Webデモです。

## 使い方

`index.html` をブラウザで開くと動作します。

## GitHub Pagesで公開する手順

1. GitHubで新しいリポジトリを作成する
2. この `webapp-var2` フォルダ内のファイルをアップロードする
3. リポジトリの `Settings` → `Pages` を開く
4. `Source` を `Deploy from a branch` にする
5. Branchを `main`、folderを `/root` にして保存する

数十秒から数分後に公開URLが表示されます。

## ファイル構成

- `index.html`: Webデモ本体
- `style-var2.css`: デザイン
- `script-var2.js`: タブ、検索、フィルターなどの動作
